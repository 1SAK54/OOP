# Task_2_4_1 - OOP Checker DSL

Консольное приложение для проверки учебных GitHub-репозиториев студентов

Конфигурация задаётся через DSL на Groovy в файле `oop-checker.groovy`

## Запуск

```bash
./gradlew run
```

Или с явным файлом конфигурации:

```bash
./gradlew run --args="examples/oop-checker.groovy"
```

Приложение печатает HTML-отчёт в стандартный вывод

## DSL

Пример конфигурации находится в папке `examples`

Поддерживается:

- `include` для подключения других файлов конфигурации
- `task` для описания задач
- `group` и `student` для описания групп
- `check` для задания проверки
- `checkpoint` для контрольных точек
- `settings` для настроек
- `bonus` для дополнительных баллов
